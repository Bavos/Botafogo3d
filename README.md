# Botafogo3d (Remotion + React Three Fiber)

Projeto de animação 3D com Remotion para renderização em MP4.

## Requisitos
- Node.js 18+

## Instalação
```bash
npm install
```

## Rodar Studio
```bash
npm run dev
```

## Renderizar MP4
```bash
npx remotion render src/index.ts Botafogo3d out/Botafogo3d.mp4
```

## Composição
- ID: `Botafogo3d`
- Duração: 10s (300 frames)
- FPS: 30
- Resolução: 1920x1080
